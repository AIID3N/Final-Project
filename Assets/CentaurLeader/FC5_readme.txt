
Fantastic Creature #5 readme.

To change the mode press:
1 - still mode (walk movement)
q - arm/disarm
2 - combat mode (run movement)
(most actions available in the combat mode)

←↑→↓ - moving
mouse 1 - attack1
mouse 2 - attack2
mouse 3 - attack3
c - trick/trick battle
u - hit1/hit2 randomly
p - defence
o - death
